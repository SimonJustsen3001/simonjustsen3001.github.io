Brobizz management

This project features an overview, analysis and implementation
of a system to manage Brobizz devices.

Overview of system

The purpose of this system is to facilitate management of BroBizz devices. Users can add, edit or delete BroBizz devices from their account as well as see the details of their trips and invoice history.

Technologies

The frontend was built with React, Typescript, Axios and MobX, while the backend was built using C# and .Net Core with Entity Framework Core.
The backend was developed with a Code-First approach using the Model-View-Controller pattern.
